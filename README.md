# 📖 Lector de Libros Web

Aplicación web para leer libros en formato **ePub** y **PDF**, creada con amor 💕

## ✨ Características

- 📚 Soporte para archivos **.epub** y **.pdf**
- 🎨 **8 temas de color** incluyendo estilo "papel antiguo" 📜
- 🔖 **Marcadores** para guardar páginas favoritas
- 📑 **Navegación por capítulos** directa
- ⛶ **Pantalla completa** para lectura inmersiva
- 🔤 Control de **tamaño y tipo de fuente**
- 💾 **Guarda tu progreso** automáticamente
- 📱 **Diseño responsive** (funciona en celular y PC)
- 🖱️ **Drag & Drop** para cargar libros
- ⌨️ Navegación con **flechas del teclado**

## 🚀 Cómo usar

1. Abre la página
2. Arrastra un archivo `.epub` o `.pdf` o haz clic para seleccionarlo
3. ¡A leer!
